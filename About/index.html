Searching Constrain Satisfaction
1. Inisialisasi:
Kelas NQueenProblem didefinisikan untuk mengelompokkan logika solusi.
Metode nQueen mengambil bilangan bulat n sebagai input, yang mewakili ukuran papan catur.

2. Struktur Data:
Tiga set (column, posDiagonal, dan negDiagonal) digunakan untuk melacak kolom dan diagonal yang sudah terisi.
res adalah array untuk menyimpan solusi akhir.
board adalah array 2D yang diinisialisasi dengan sel-sel papan kosong.

3. Fungsi Backtracking (backtrack):
Logika penyelesaian utama diimplementasikan dalam fungsi backtrack, yang mengeksplorasi konfigurasi yang berbeda.
Fungsi ini rekursif dan mengambil baris saat ini (r) sebagai argumen.

4. Base Case:
Jika baris saat ini (r) mencapai jumlah total baris (n), sebuah solusi ditemukan, dan konfigurasi papan saat ini ditambahkan ke hasil (res).

5. Iterasi Melalui Kolom:
Fungsi ini melakukan iterasi melalui setiap kolom dalam baris saat ini (c).
Jika ratu dapat ditempatkan pada posisi saat ini tanpa bertentangan dengan ratu yang sudah ada, ratu ditempatkan di papan, dan set diperbarui sesuai.

6. Rekursi:
Fungsi kemudian dipanggil secara rekursif untuk baris berikutnya (r + 1).
Proses ini terus berlanjut sampai solusi valid ditemukan untuk seluruh papan.

7. Backtracking:
Setelah mengeksplorasi semua kemungkinan untuk posisi saat ini, ratu yang ditempatkan dihapus, dan set dikembalikan ke keadaan sebelumnya untuk backtracking.
Ini memungkinkan algoritma mengeksplorasi konfigurasi lainnya.

8. Hasil:
Fungsi melanjutkan proses ini sampai semua solusi yang mungkin dieksplorasi.
Hasil akhir berupa array konfigurasi papan yang valid.

9. Mengembalikan Hasil:
Array res, yang berisi semua solusi valid, dikembalikan.
Display to HTML
Kode ini mengintegrasikan pemecahan masalah N-Queens dengan tampilan HTML. Hasilnya ditampilkan dalam bentuk setiap solusi dengan nomor solusi yang sesuai. Juga, menangani kasus khusus di mana solusi tidak ditemukan untuk ukuran papan tertentu.
